# HR_analytics_dashboard
The dashboard provides a comprehensive view of HR metrics including employee count, attrition rate, average salary, average age, and tenure. It also explores attrition patterns across departments, gender, education levels, salary bands, age groups, job roles, and years at the company.

Using Power BI, I built an interactive dashboard that enables HR teams to quickly identify areas where employee churn is highest and where retention strategies may be needed.

Through this project, I applied data modeling techniques and DAX calculations to derive meaningful HR metrics and designed clear visualizations to communicate insights effectively.

This project demonstrates how data analytics can support HR decision-making by transforming employee data into actionable insights that help improve retention, workforce planning, and organizational performance.

Technologies used: Power BI, DAX, Data Modeling, Data Visualization
